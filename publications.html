Kai-Xuan Chen - Papers

Publications (Journal Papers)

 

  • K.-X. Chen, J.-Y. Ren, X.-J. Wu*, J. Kittler. Covariance Descriptors on a Gaussian Manifold and their Application to Image Set Classification[J]. Pattern Recognition, 2020: 107463. [paper][code]

    • To generate Riemannian covariance descriptors, we present two methods to calculate a Riemannian local difference vector on the Gaussian manifold and extend the conventional covariance computation onto a this special type of Riemannian manifold.

 

  • R. Wang, X.-J. Wu*, K.-X. Chen, J. Kittler. Multiple Riemannian Manifold-valued Descriptors based Image Set Classification with Multi-Kernel Metric Learning[J]. IEEE Transactions on Big Data, 2020. in press.[paper] [code]

    • To fuse multiple heterogeneous Riemannian Manifold-valued features, we devise a multi-kernel metric learning framework to embed them into a lower dimensional common subspace.

 

  • K.-X. Chen, X.-J. Wu*. Component SPD matrices: A low-dimensional discriminative data descriptor for image set classification[J]. Computational Visual Media, 2018, 4(3): 245-252. [paper] [code]

    • A simple but very effective framework to generate low-dimensional discriminative representation with application to image set classification.

  • 陈凯旋, 吴小俊. 基于对称正定流形潜在稀疏表示分类算法 [J]. 软件学报, 2020,31(8):2530-2542.

  • 陈凯旋, 吴小俊. 面向图像集分类的切空间稀疏表示算法 [J]. 中国图象图形学报, 2018, 23(7): 961-972.

Publications (Conference Papers)

 

  • K.-X. Chen, X.-J. Wu*, J.-Y. Ren, R. Wang, J. Kittler. More About Covariance Descriptors for Image Set Coding: Log-Euclidean Framework based Kernel Matrix Representation[C]. IEEE International Conference on Computer Vision Workshop (ICCVW), The 4th CEFRL, 2019.[paper] [code]

    • An improved version of CSPD for image set coding by considering the Log-Euclidean framework based arc-cosine (LogE.Arc) kernel.

 

  • K.-X. Chen, X.-J. Wu*, R. Wang, J. Kittler. Riemannian kernel based Nyström method for approximate infinite-dimensional covariance descriptors with application to image set classification[C]/ IEEE International Conference on Pattern Recognition (ICPR), 2018: 651-656.[paper] [code]

    • To overcome the shortcomings of infinite-dimensional covariance descriptors, we propose to represent image sets by approximating them in the paradigm of the Nyström method based on a Riemannian kernel.

 

  • R. W., X.-J. Wu*, K.-X. Chen, J. Kittler. Multiple Manifolds Metric Learning with Application to Image Set Classification[C]. IEEE International Conference on Pattern Recognition (ICPR), 2018: 627- 632. [paper] [code]

    • Motivated by the inability of existing methods to extract features for data on Riemannian manifolds, we propose a novel algorithm which combines multiple manifolds as the features of the original image sets.
